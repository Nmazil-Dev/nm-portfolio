Weekly Jams

Description

Using EDMTrain and Spotify Apis, a user can listen the top songs from local djs playing a show the current week, view event details, and connect to Spotify to get their weekly edm playlist added to their Spotify account.

Developed With

  • Flask
  • Python
  • SQLite
  • Javascript
  • Bootstrap4
  • Spotify and EDMTrain Apis

Purpose

Weekly Jams allows a user to discover new music from local djs in their state, and listen to music they can expect to hear at music venues that week using Spotify.

My Libly

Description

Using a search engine powered by GoogleBooks Api, users can search books which they are currently reading in order to track their reading progress by page count via a progress bar. Users can also create accounts in order to build their own library.

Developed With

  • Django
  • Python
  • Javascript
  • Bootstrap4
  • Google Books Api

Purpose

My Libly allows users to track their reading progress of print books via a progress bar and encourages users to complete books which they have started but have not finished.
